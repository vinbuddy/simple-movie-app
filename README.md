# Simple Movie App

![App](/src/assets/images/SMA.png)

## Some technology used

-   ReactJS
-   React-hook-form
-   React-bootstrap
-   React-tippy
-   Firebase
-   Swiper
-   Axios

## API

-   The New York times API
-   TMDB API

## Features

-   Movie news
-   Search film
-   Explore film
-   Sort, Filter
-   Trailer
-   Authentication (sign in / sign up)
-   Save and share film
-   Create collection (create, edit, delete collection)
-   Edit profile (avatar, name)
-   Toggle dark/light mode
-   404 page
